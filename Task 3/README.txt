Para permitir a intereção dos usuários com os jobs pode-se definir que o job é
parametrizado.

Para soluciar o desáfio da task 3 poderiamos criar uma pipeline parametrizada,
onde o parametro poderia vir até de uma fonte externa, um exemplo de fonte
externa seria a interação com o slack. O parâmetro passado seriá a versão da
aplicação que desejamos realizar o deploy. Dentro da pipeline esse parâmetro
poderia ser passado para uma várivel e usado no código.
